Introduction to HTML

HTML, or HyperText Markup Language, is the foundation of all web pages. It defines the structure and content of a webpage using tags and elements. Every web browser reads HTML and displays it in a human-readable format.

Key Features of HTML

  • It is a markup language — not a programming language.
  • Uses tags enclosed in angle brackets (< >).
  • Supports multimedia like images, audio, and video.
  • Is platform-independent and works on all browsers.

Structure of an HTML Page

<!DOCTYPE html> <html lang="en"> <head> <title>My First Page</title> </head> <body> <h1>Welcome to Web Development</h1> <p>HTML is the backbone of the web!</p> </body> </html>

How HTML Works

When a browser loads an HTML file, it reads the markup from top to bottom and converts it into a Document Object Model (DOM). CSS and JavaScript can then modify how this structure looks and behaves.

HTML Elements

HTML elements are the building blocks of web pages. They represent parts of the document such as headings, paragraphs, images, and links. Each element generally has an opening tag, content, and a closing tag.

Common Elements

  • <h1>–<h6>: Headings from largest to smallest.
  • <p>: Paragraph of text.
  • <a>: Anchor element for hyperlinks.
  • <img>: Image element (self-closing).
  • <div>: Container for grouping elements.
  • <span>: Inline container for styling text.

Example

<div> <h2>My Favorite Quote</h2> <p>"The best way to get started is to quit talking and begin doing."</p> <a href="https://example.com">Learn more</a> </div>

Elements can also be nested, meaning you can place one element inside another. This helps organize your HTML structure efficiently.

Block vs Inline Elements

  • Block elements start on a new line (e.g., <div>, <p>, <h1>).
  • Inline elements stay within the same line (e.g., <span>, <a>, <img>).

HTML Attributes

Attributes provide additional information about HTML elements. They always appear inside the opening tag and consist of a name/value pair like name="value".

Common Attributes

  • href — Defines the destination of a hyperlink.
  • src — Specifies the source of an image, video, or audio file.
  • alt — Provides alternative text for an image.
  • id — Gives a unique identifier to an element.
  • class — Assigns one or more class names for styling.
  • title — Displays extra info when hovering over the element.

<a href="https://www.wikipedia.org" title="Visit Wikipedia">Wikipedia</a> <img src="flower.jpg" alt="A beautiful flower" width="300">

Global Attributes

These attributes can be used on any HTML element:

  • style — Adds inline CSS styles.
  • hidden — Hides the element from view.
  • tabindex — Specifies the tab order for keyboard navigation.
  • contenteditable — Makes text editable directly in the browser.

HTML Forms

Forms are used to collect user input. They can include text fields, checkboxes, radio buttons, dropdowns, and submit buttons.

Example

<form action="/submit" method="post"> <label>Name:</label> <input type="text" name="name" required><br><br> <label>Email:</label> <input type="email" name="email" required><br><br> <label>Gender:</label> <input type="radio" name="gender" value="male"> Male <input type="radio" name="gender" value="female"> Female<br><br> <input type="submit" value="Submit"> </form>

Common Input Types

  • text, password, email
  • checkbox, radio, range
  • date, file, color
  • submit, reset, button

HTML5 Form Enhancements

  • required — Ensures the field must be filled before submission.
  • placeholder — Displays hint text inside inputs.
  • pattern — Allows input validation with regex.

HTML Media

Media elements like images, audio, and video make web pages visually appealing and interactive. HTML5 introduced powerful tags for embedding multimedia content.

Examples

<img src="nature.jpg" alt="Nature" width="400"> <audio controls> <source src="song.mp3" type="audio/mpeg"> Your browser does not support the audio tag. </audio> <video width="480" controls> <source src="sample.mp4" type="video/mp4"> Your browser does not support the video tag. </video>

HTML5 supports attributes like autoplay, loop, and muted for controlling media behavior. It also allows adding captions through <track> elements for accessibility.

HTML Semantics

Semantic HTML gives meaning to content, helping browsers, developers, and search engines understand its purpose. It improves SEO and accessibility for screen readers.

Common Semantic Tags

  • <header> — Defines the top section or heading.
  • <nav> — Contains navigation links.
  • <article> — Represents self-contained content like blog posts.
  • <section> — Groups related content.
  • <aside> — Sidebar content or ads.
  • <footer> — Defines footer section.

<header> <h1>My Blog</h1> </header> <article> <h2>First Post</h2> <p>This is my first blog post using semantic HTML.</p> </article> <footer>© 2025 My Blog</footer>

Using semantic elements makes your HTML cleaner and more descriptive. It’s considered a best practice in modern web development.

CSS Development

1. CSS Selectors

CSS selectors define which HTML elements your styles will apply to. They are the foundation of CSS design.

  • Universal Selector: * { margin: 0; padding: 0; }
  • Type Selector: p { color: blue; }
  • Class Selector: .box { background: yellow; }
  • ID Selector: #main { border: 2px solid red; }
  • Attribute Selector: input[type="text"] { border-radius: 5px; }

Selectors can be combined for complex targeting. For example, div p applies styles to paragraphs inside divs only.

2. CSS Colors

CSS allows you to apply colors using names, HEX, RGB, RGBA, or HSL values.

  • Named Color: color: red;
  • HEX Value: color: #ff0000;
  • RGB: color: rgb(255,0,0);
  • RGBA (with transparency): color: rgba(255,0,0,0.5);
  • HSL: color: hsl(0, 100%, 50%);

Colors are not limited to text; you can color borders, backgrounds, and shadows as well.

3. CSS Backgrounds

Backgrounds enhance the visual appeal of web pages by applying colors, gradients, or images.

Examples:

  • background-color: lightblue;
  • background-image: url('bg.jpg');
  • background-repeat: no-repeat;
  • background-size: cover;
  • background-position: center;

CSS gradients allow smooth transitions between colors using linear-gradient() or radial-gradient().

4. CSS Borders

Borders help visually separate or highlight elements.

Examples:

  • border: 2px solid black;
  • border-radius: 10px;
  • border-style: dotted dashed solid double;

Rounded borders give a modern, soft look to containers.

5. CSS Box Model

The box model explains how every element is a rectangular box with four layers: content, padding, border, and margin.

div { width: 200px; padding: 20px; border: 5px solid black; margin: 15px; }

Understanding the box model helps fix layout issues and design responsive pages.

6. CSS Shadows

Shadows add depth and realism to designs. CSS supports both text and box shadows.

Examples:

  • Box Shadow: box-shadow: 4px 4px 15px rgba(0,0,0,0.3);
  • Text Shadow: text-shadow: 2px 2px 5px gray;

Shadows can create soft glowing effects, 3D designs, or floating element visuals.
